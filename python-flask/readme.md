# AI Food Delivery Using Drones (Python Flask)

This is a simulation project for **AI-based drone food delivery** with:

- Flask API backend
- SQLAlchemy models
- Drone assignment logic (distance + battery + radius)
- Real-time drone movement using Socket.IO
- Live map using Leaflet + OpenStreetMap
